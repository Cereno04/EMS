##  EMS Admin - Employee Management System EMS Admin - Employee Management System
A robust, full-stack administrative platform built with Laravel to streamline human resource operations. This system allows administrators to manage workforce data, track departmental structures, and monitor real-time organizational statistics through a clean, modern dashboard.

## Key Features
Administrative Dashboard: Real-time visualization of total employees, active staff, and departmental counts.
Employee Management (CRUD): Full capability to onboard, update, and manage employee profiles, including roles and contact information.
Departmental Organization: Dedicated module to manage company departments and assign managers.
Attendance Tracking: Built-in sidebar integration for future attendance logging.
Responsive UI: Crafted with a "Mobile-First" approach using modern CSS components for a seamless experience across all devices.

## Tech Stack
- Backend: Laravel (PHP)
- Frontend: Tailwind CSS / Blade Templating
- Database: SQLite
- Tools: VScode, Herd
